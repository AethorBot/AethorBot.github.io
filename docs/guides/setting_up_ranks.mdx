### Malil has free to use ranks

![ranks](../../static/img/guides/rank/ranks.png)

### How to set them up?

you can use the slash command `/addrank @role` or you can use the normal command `*addrank @role/roleId/roleName`

![ranks](../../static/img/guides/rank/addrank.png)

#### Joining ranks

Just do `/rank @role` or the normal `*rank` command  
Join the rank again to leave it
